        <H2>contoh dan fungsi elemen HTML</H1>
        <p>
            <hr>
            1. Struktur Dokumen : <br>
            - html: Elemen root dari dokumen HTML, membungkus seluruh konten HTML.<br>
            - head: Bagian untuk metadata, pengaturan, dan elemen-elemen seperti judul halaman (title), link ke CSS, dan script.<br>
            - body: Berisi konten utama halaman web, seperti teks, gambar, dan elemen interaktif.<br>
            <hr>
            2. Elemen Teks : <br>
            - h1 hingga h6: Digunakan untuk judul (heading) dengan h1 sebagai yang paling penting dan besar, hingga h6 sebagai yang paling kecil.<br>
            - p: Menandai sebuah paragraf.<br>
            - span: Elemen inline untuk menandai bagian kecil dari teks tanpa mengubah tata letak.<br>
            - strong: Menandai teks yang penting (secara default ditampilkan tebal).<br>
            - em: Menandai teks yang perlu diberi penekanan (secara default ditampilkan miring).<br>
            - b: Menebalkan teks tanpa memberikan penekanan semantik.<br>
            - i: Memiringkan teks tanpa memberikan penekanan semantik.<br>
            - mark: Menandai teks yang relevan atau penting (misalnya, highlight).<br>
            - abbr: Menandai singkatan atau akronim.<br>
            - cite: Menandai referensi atau kutipan dari sumber lain.<br>
            - code: Menampilkan teks sebagai kode pemrograman.<br>
            - pre: Menampilkan teks dalam format preformatted, menjaga spasi dan baris baru.<br>
            - q: Kutipan singkat dalam satu baris.<br>
            - blockquote: Kutipan panjang dari sumber lain.<br>
            - sub: Menampilkan teks bertanda bawah (subscript).<br>
            - sup: Menampilkan teks bertanda atas (superscript).<br>
            - kbd: Menandai input dari keyboard pengguna.<br>
            - samp: Menampilkan hasil dari program atau perangkat lunak.<br>
            - var: Menandai sebuah variabel dalam kode pemrograman.<br>
            - small: Menampilkan teks dengan ukuran lebih kecil.<br>
            - del: Menandai teks yang dihapus atau diubah.<br>
            - ins: Menandai teks yang disisipkan atau ditambahkan.<br>
            - u: Menandai teks yang digarisbawahi.<br>
            <hr>
            3. Link & Media : <br>
            - a: Membuat hyperlink untuk menghubungkan ke halaman lain atau sumber lain.<br>
            - img: Menyisipkan gambar ke dalam halaman web.<br>
            - audio: Menyisipkan file audio untuk diputar di halaman web.<br>
            - video: Menyisipkan file video yang dapat diputar di halaman web.<br>
            - source: Memberikan sumber alternatif untuk elemen media seperti audio dan video.<br>
            - embed: Menyematkan objek multimedia atau aplikasi eksternal seperti Flash, SVG, atau video.<br>
            - object: Menyematkan objek multimedia atau aplikasi eksternal, misalnya file PDF atau Flash.<br>
            - iframe: Menyematkan halaman HTML lain dalam halaman web ini.<br>
            - area: Menentukan area interaktif dalam gambar (untuk membuat peta gambar).<br>
            <hr>
            4. Formulir : <br>
            - form: Membuat elemen formulir untuk menerima input dari pengguna.<br>
            - input: Elemen untuk berbagai jenis input form (misalnya teks, tombol radio, checkbox).<br>
            - textarea: Area untuk menulis teks panjang.<br>
            - button: Membuat tombol yang bisa diklik untuk melakukan aksi.<br>
            - select: Dropdown menu untuk memilih satu atau lebih opsi.<br>
            - option: Menentukan opsi dalam dropdown menu.<br>
            - label: Memberikan label untuk elemen form, seperti input atau select.<br>
            - fieldset: Mengelompokkan elemen-elemen form dalam satu grup.<br>
            - legend: Memberikan judul atau deskripsi untuk grup yang dikelompokkan dalam fieldset.<br>
            - input type="checkbox": Checkbox untuk memilih satu atau beberapa opsi.<br>
            - input type="radio": Radio button untuk memilih hanya satu dari beberapa opsi.<br>
            - input type="file": Input untuk mengunggah file.<br>
            - input type="range": Input berbentuk slider untuk memilih nilai dalam rentang.<br>
            - input type="color": Input untuk memilih warna.<br>
            - input type="date": Input untuk memilih tanggal.<br>
            - input type="email": Input untuk memasukkan email.<br>
            - input type="password": Input untuk memasukkan password yang disembunyikan.<br>
            - input type="number": Input untuk memasukkan angka.<br>
            - input type="tel": Input untuk nomor telepon.<br>
            - input type="url": Input untuk memasukkan URL.<br>
            - input type="search": Input untuk pencarian teks.<br>
            <hr>
            5. Tabel : <br>
            - table: Membuat tabel.<br>
            - tr: Baris dalam tabel.<br>
            - td: Sel tabel yang berisi data.<br>
            - th: Sel tabel yang berisi judul kolom atau baris.<br>
            - caption: Menyediakan judul atau keterangan untuk tabel.<br>
            - col: Menentukan kolom dalam tabel.<br>
            - colgroup: Mengelompokkan kolom dalam tabel.<br>
            - thead: Menandai bagian atas tabel (kepala tabel).<br>
            - tbody: Menandai bagian tubuh tabel (konten tabel).<br>
            - tfoot: Menandai bagian bawah tabel (kaki tabel).<br>
            <hr>
            6. List : <br>
            - ul: Daftar tidak terurut (bullet points).<br>
            - ol: Daftar terurut (nomor).<br>
            - li: Item dalam daftar.<br>
            - dl: Daftar definisi.<br>
            - dt: Istilah dalam daftar definisi.<br>
            - dd: Definisi dari istilah dalam daftar definisi.<br>
            <hr>
            7. Pengelompokan dan Layout : <br>
            - div: Elemen blok untuk mengelompokkan konten dan membagi halaman menjadi bagian-bagian.<br>
            - section: Menandai bagian atau segmen dari halaman dengan topik tertentu.<br>
            - article: Konten yang bisa berdiri sendiri, seperti artikel atau posting blog.<br>
            - header: Bagian atas dari halaman atau artikel (biasanya berisi judul atau navigasi).<br>
            - footer: Bagian bawah dari halaman atau artikel (biasanya berisi informasi tambahan atau hak cipta).<br>
            - main: Konten utama dari halaman.<br>
            - aside: Konten sampingan yang tidak terlalu relevan dengan bagian utama.<br>
            - nav: Elemen untuk menu navigasi (link ke bagian lain dari halaman atau situs).<br>
            - figure: Menyisipkan gambar atau media lainnya dengan keterangan.<br>
            - figcaption: Memberikan keterangan untuk elemen [figure].<br>
            - details: Menyediakan konten yang dapat dibuka atau disembunyikan pengguna.<br>
            - summary: Memberikan ringkasan untuk elemen [details].<br>
            - mark: Menandai teks yang relevan atau penting (highlighted).<br>
            - output: Menampilkan hasil dari kalkulasi atau perhitungan.<br>
            - progress: Menampilkan progres dari suatu proses.<br>
            - meter: Menampilkan nilai dalam rentang tertentu (misalnya, level baterai).<br>
            - template: Menyimpan HTML yang tidak langsung di-render, untuk digunakan nanti.<br>
            - slot: Digunakan dalam Web Components untuk menyisipkan konten dari luar.<br>
            - wbr: Menyisipkan kemungkinan break line dalam kata panjang.<br>
            - hr: Membuat garis horizontal untuk memisahkan konten.<br>
            - br: Break line, untuk membuat baris baru dalam teks.<br>
            <hr>
            8. Elemen Metadata : <br>
            - meta: Menyediakan metadata seperti deskripsi halaman, pengaturan karakter, dll.<br>
            - title: Menentukan judul halaman yang ditampilkan di tab browser.<br>
            - link: Menyambungkan dokumen HTML dengan file eksternal seperti CSS.<br>
            - base: Menentukan URL dasar untuk link di dalam dokumen.<br>
            - style: Menyisipkan CSS langsung dalam dokumen HTML.<br>
            - script: Menyisipkan JavaScript untuk interaktivitas.<br>
        </p>
